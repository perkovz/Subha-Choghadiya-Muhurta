# Subha Choghadiya Muhurta

A simple, beautiful Chrome extension that shows today's **Choghadiya Muhurta** timings (auspicious & inauspicious periods) + **Rahu Kaal**, **Abhijit Muhurat**, **Kaal Vela / Vaar Vela**, and **Kaal Ratri** ‚Äî all calculated using real sunrise/sunset for your chosen city. We only use local storage (to remember city) and make a single public API call for sunrise/sunset, no personal data collected Limited number of cities for now but adding more every week.

### Features
- Accurate Choghadiya periods (day & night) based on Drik Panchang-inspired sequences
- Highlights current active period with ‚è≥ icon
- Shows Rahu Kaal in a dedicated section
- Includes Abhijit Muhurat (auspicious midday window) at the top
- Supports 11 cities across USA, Canada, Croatia, and India (easy to add more)
- Automatic timezone & DST handling ‚Äî works correctly when you travel or switch cities
- Clean, glanceable design with color-coded good/bad periods

### How to use
1. Click the extension icon
2. Choose your city from the dropdown
3. See today's timings instantly
4. The ‚è≥ shows which Choghadiya period is active right now (in the selected city's local time)
