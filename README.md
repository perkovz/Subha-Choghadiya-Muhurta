# Subha Choghadiya Muhurta

A simple, Chrome extension that shows today's **Choghadiya Muhurta** timings (auspicious & inauspicious periods) + **Rahu Kaal**, **Abhijit Muhurat**, **Kaal Vela / Vaar Vela**, and **Kaal Ratri** â€” all calculated using the real sunrise/sunset for your city. No personal data collected is collected.

## Features
- Current daily Choghadiya periods
- Highlights current active Muhurta
- Shows Abhijit Muhurat & Rahu Kaal in a dedicated sections

## How to use
1. Click the extension icon
2. Search for the closest city
3. Timings for the chosen location will update instantly
